# 📊 CX360 – Agent Performance & Customer Analytics

## 🚀 Overview
CX360 is a data-driven platform designed to analyze **agent performance** and improve **customer experience** across multiple channels like calls, emails, and social media.

It automates performance evaluation, reducing dependency on manual QA reviews and enabling faster, data-driven decisions.

---

## 🎯 Key Features
- Omnichannel data processing  
- Sentiment & tone analysis  
- Intent recognition  
- Agent scorecards  
- KPI tracking (AHT, CSAT)  
- Power BI dashboards  

---

## 🏗️ Architecture (Medallion)

- **Bronze Layer**: Raw data ingestion via APIs, stored in data lake  
- **Silver Layer**: Data validation (Pydantic), cleaning & transformation  
- **Gold Layer**: Business transformations (CDC, SCD), data warehouse  

---

## 📊 Visualization
- Interactive dashboards built using **Power BI**  
- Tracks agent performance and customer insights  

---

## 🛠️ Tech Stack
- Python, Pydantic  
- Apache Spark / Databricks  
- AWS S3 (Data Lake)  
- Data Warehouse  
- Power BI

## 📈 Impact
- Automated agent evaluation  
- Improved efficiency & productivity  
- Better customer satisfaction
